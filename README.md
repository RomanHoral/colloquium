# Агрегатор цін на авіаквитки

## Опис
Система для пошуку та порівняння цін на авіаквитки з різних джерел.

## Функціонал
- Пошук рейсів
- Фільтрація за ціною, часом та пересадками
- Сортування результатів

## Встановлення

### Запуск через Docker
```bash
docker build -t flights-app .
docker run -p 5000:5000 flights-app
